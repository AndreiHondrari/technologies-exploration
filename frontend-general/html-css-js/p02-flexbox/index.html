Formula for flex-shrink

Let's assume we have:

  • a container with display: flex that has two elements
  • the main container has a fixed width of 200px
  • both sub-elements are set to 150px width
  • 1st element with flex-shrink: 1
  • 2nd element with flex-shrink: 3

Obtain the outstanding difference:

a = (150px + 150px) - 200px = -100px

Calculate the total flex parts and distribute the outstanding difference to all parts equally.

b = 1 + 3 = 4
c = -100px / b = -25px

Subtract from their actual widths the weighted amount:

elem1_width = 150px + -25px * 1 = 125px
elem2_width = 150px + -25px * 3 = 75px

Basically what this formula says is: subtract from each element as much as their shrink is worth


Test 8

fb auto, width 75%
.a flex-shrink = 0

Basically:

width_difference = (75% + 75%) - 100% = -50%
a_parts + b_parts = 1
diff_per_part = -50% / 1 = -50%
a_width = 75% + (0 * (-50%)) = 75%
b_width = 75% + (1 * (-50%)) = 25%

.a flex-shrink = 2 .b flex-shrink = 1 (default)

Basically:

width_difference = (75% + 75%) - 100% = -50%
a_parts + b_parts = 4
diff_per_part = -50% / 4 = -12.5%
a_width = 75% + (3 * -12.5%) = 37.5%
b_width = 75% + (1 * -12.5%) = 62.5%
